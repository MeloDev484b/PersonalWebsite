My Projects

Scheduling App - Software II

This software is capable of adding, removing, and modifying customer and appointment records, and includes localization for French and English. I was provided with an Entity-Relationship digram and tasked with designing the GUI, and UML diagram before building the application. This scheduling application is written in Java, built with JavaFX, and utilizes the JDBC API to store data in the user's database.

Inventory Management System - Software I

I was tasked with prototyping a GUI application with a provided UML diagram, a storyboard and a list of desired functionalities. This software is capable of adding, removing, and modifying parts and products. When a product is added, parts are associated with that product. This inventory management system was written in Java and built with JavaFX.

Building Game - Personal project

Building Game is a 2D resource-gathering simulation. Currently, it is a work-in-progress that I have been using to learn about game design and development. Building Game currently features a building system that validates construction, starving mechanics that cause citizens to leave if they are not fed, 6 buildings with different behaviors, and various panels for the player to interact with. It is written in GDScript and built with the Godot game engine.
